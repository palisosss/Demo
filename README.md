# Urban Gear — вариант 2 (модули 1–4)

Отдельный вариант desktop-приложения на `Python + Tkinter + SQLite`.

## Что реализовано

- **Модуль 1**: нормализованная БД, внешние ключи, скрипт структуры.
- **Модуль 2**: авторизация (гость/клиент/менеджер/админ), каталог товаров.
- **Модуль 3**: поиск/фильтрация/сортировка + CRUD товаров (админ).
- **Модуль 4**: экран заказов для менеджера/админа, CRUD заказов (админ).

## Запуск

```bash
python app_v2.py
```

## Тестовые пользователи

- `root` / `root123` — администратор
- `boss` / `boss123` — менеджер
- `buyer` / `buyer123` — клиент
